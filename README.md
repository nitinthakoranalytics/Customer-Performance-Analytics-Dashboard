# Customer Performance Analytics Dashboard (Power BI)

This Power BI dashboard analyzes customer sales performance using key KPIs, sales trends, top customers, and regional breakdown.

---

## Project Highlights
- Built an interactive dashboard to track **Total Sales, Total Customers, Total Transactions**, and **Average Sales per Customer**
- Created time-series analysis to understand **sales trends over time**
- Added region-level comparison to identify **highest performing regions**
- Highlighted top customers to support **customer targeting and retention decisions**

---

## Tools Used
- Power BI
- DAX Measures
- Data Modeling & Relationships
- Filters / Slicers
